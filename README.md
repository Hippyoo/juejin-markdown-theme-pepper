## 颜色使用说明


| 文字效果      |           编写方式           |           使用方式            |
| ------------- | :--------------------------: | ----------------------- |
| ~~删除线~~   | ~~删除线~~ `or` <s>标签包裹</s> |  做正常的删除线使用|
| *斜体字*     |    \* 斜体字\* `or` \_斜体字\_     | 标题中的重点文字|
| **粗体字**   |    \*\*粗体\*\* `or` \_\_粗体\_\_    |   正文段落中的重点文字|
| ***粗斜体*** | \*\*\*粗斜体\*** `or` \_\_\_粗斜体\_\_\_ |  正文段落中的小标题“xx：”|
